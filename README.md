Bob's Burgers "Snake" Game
==========================

Feed Bob Burgers!

Use the arrow keys to move towards the burger on the screen.

Watch as Bob's head multiplies as he eats his burgers!

## Getting Started

### Requirements :key:

* Python 3.4+
  - [Pygame](https://www.pygame.org/wiki/GettingStarted)

### Installing :floppy_disk:

* pip install -r requirements.txt

### Usage :computer:

* Simply run, and the game begins
```
python snake.py
```

## Notes

This game is very minimal, an exercise to try out pygame. The Bob snake will not bend as it grows longer and there is no score count.
